Xadmin安装

  • 安装步骤：
    1. 前往github，搜索xadmin，寻找是否有一些作者对xadmin的兼容性进行修复，可参考此链接。
    2. 将其clone至本地，可选择运行里面的setup.py进行安装。
    3. 如果没有setup.py，可先安装旧版本xadmin，删除site-packages目录下的xadmin目录，之后将更改兼容性后的xadmin打包文件夹改名后仿佛site-packages目录下。
    4. 进入Python环境后输入import xadmin不报错即证明安装成功。
